# Budgeting App

A modern personal budgeting application that helps you manage recurring expenses, income, budgets, savings goals, and financial reports in one place.

## Features
- Dashboard with spending, remaining budget, savings rate, and net income.
- Manage recurring expenses with flexible schedules.
- Organise expenses into categories and groups.
- Track income sources and estimated take-home pay.
- Create savings goals.
- Monthly budgets with overspending alerts.
- Reports and trends across months.
- Import existing spreadsheet data.
- Export data to CSV/Excel.

## Getting Started
1. Sign in with your Google account.
2. Add or customise your expense categories.
3. Add your income source(s).
4. Enter recurring and one-off expenses.
5. Review the dashboard to monitor your remaining budget.
6. Set budgets and savings goals.
7. Explore reports to analyse spending over time.

## Main Sections
- **Dashboard** – Financial overview.
- **Expenses** – Create and manage expenses.
- **Budgets** – Set category budgets.
- **Reports** – View trends and charts.
- **Goals** – Track savings goals.
- **Settings** – Manage profile, tax settings, and preferences.

## Recurring Expenses
Choose a frequency (daily, weekly, fortnightly, monthly, quarterly, yearly, or custom), with optional start and end dates.

## Data
The application stores your income, expenses, categories, and goals, then calculates totals, budgets, and reports automatically.

## Roadmap
Future updates include receipt uploads, shared household budgets, bank imports, and AI-powered insights.
